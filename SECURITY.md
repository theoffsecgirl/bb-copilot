# Security Policy

## Supported versions

| Version | Supported |
|---------|----------|
| 1.x     | ✅ Yes    |

## Reporting a vulnerability

If you find a security issue in this tool itself, please report it responsibly.

**Do not open a public issue.**

Contact: [info@theoffsecgirl.com](mailto:info@theoffsecgirl.com)  
Include: description, reproduction steps, potential impact.

Expected response time: **72 hours**.

## API key handling

Never commit your `.env` file. It is listed in `.gitignore` by default.  
Use `.env.example` as a template.
